Introduction to Positions Css Modes:

Sticky:

Sticky Position: sticks on the page if u scroll up or down it will stay on the page wherever u go it will stick

Static:

Static Position: is the default value. An element with position static is not positioned in any special way. A static element is said to be not positioned and an element with its position set to anything else is said to be positioned.

Relative:

Relative Position: behaves the same as static unless you add some extra properties also it sticks to the before and after element and you can also put in it absolute position mode.

Setting the top, right, bottom, and left properties of a relatively-positioned element will cause it to be adjusted away from its normal position. Other content will not be adjusted to fit into any gap left by the element.

Absolute:

Absolute Position: can be put wherever you want over elements or sideways or top bottom above everywhere u desire, absolute becomes static if there was none elements with position before it, unless there was an element positioned before it the element is positioned absolutely to its first positioned parent.

Fixed:

Fixed Position: sticks on the page if u scroll up or down it will stay on the page wherever u go it will stick.
